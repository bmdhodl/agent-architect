# Changelog

All notable changes to Agent Architect will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] — 2026-05-01

### Added
- Initial release.
- `scope-agent` skill that scopes and cost-estimates an AI agent build from a plain English description.
- Tier classification: DIY ($500–$2K) / Startup ($2K–$15K) / Growth ($15K–$50K) / Enterprise ($50K–$150K).
- Output report with cost breakdown, architecture, timeline, and top 3 risks.
- Free tier counter (5 scopes per calendar month) tracked at `~/.agent-architect/usage.json`.
- CTA to bmdpat.com/audit for Startup-tier scopes and above (UTM-tagged).
